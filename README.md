## ğŸ¯ Customization

### Changing Images
Replace the placeholder URLs in `index.html`:
```html
<!-- Institution Logo -->
<img src="YOUR_LOGO_URL_HERE" alt="Logo">

<!-- Profile Picture -->
<img src="YOUR_PROFILE_PICTURE_URL_HERE" alt="Profile Picture">
```

### Updating Information
Edit the content in `index.html`:
- Change institution name in the header section
- Update all student information fields
- Modify emergency contact details
- Customize footer text

### Modifying Colors
Edit the color values in `style.css`:
- Primary: `#A10605` (default red)
- Secondary: `#050DA1` (default blue)
- Accent: `#FFC600` (default yellow)

### Changing Font
Replace the Google Fonts link in `index.html` and update `font-family` in `style.css`.
